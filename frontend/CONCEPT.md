# F1 Dashboard

## What is this?
このシステムはF1のレースデータをダッシュボード化し、ユーザーが自由に分析できるインタラクティブなWebアプリケーションです。Open F1 APIをデータソースとして、リアルタイムおよび過去のレース情報を可視化します。

## ユーザーニーズ
- **観戦体験の深化**: 自分でデータをフィルタリングし詳細を深掘りすることで、「この時の状況はこうだった」「このラップで一気にタイヤがなくなったんだ！」などの発見をもたらします
- **戦略理解の促進**: ピット戦略やタイヤマネジメントの意図を可視化することで、チームの戦略を理解できます
- **パフォーマンス比較**: ドライバーやチーム間のパフォーマンス差を明確に把握できます
- **レース後の振り返り**: レース後に詳細なデータで振り返ることで、より深い理解と楽しみを提供します

## 主な機能

### 本戦（Race）分析
#### 1. ラップタイムとピットタイミングの可視化
- **ラップタイム推移グラフ**
  - 全ドライバーのラップタイムを時系列で表示
  - 特定ドライバーの選択・比較機能
  - どこで突き放されたのか、追いついたのかを視覚的に確認
- **ピット戦略の可視化**
  - アンダーカット・オーバーカットの成功/失敗の判定と可視化
  - ピットストップ前後のラップタイム変化
  - ピットストップのタイミングとロスタイムの表示
- **タイヤパフォーマンス分析**
  - タイヤコンパウンド（ソフト/ミディアム/ハード）別の平均ラップタイム
  - タイヤ摩耗によるデグラデーション（ラップごとのタイム劣化）の可視化
  - スティント（ピット間の走行区間）ごとのパフォーマンス比較

#### 2. セクション・ターンごとの速度分析
- **セクタータイム比較**
  - セクター1/2/3の各タイムをドライバー間で比較
  - どのセクターで速かった/遅かったかを色分け表示
- **ミニセクター速度分析**
  - より細かい区間（ターン単位）での速度データ
  - コーナリング速度とストレート速度の分離表示
- **ヒートマップ表示**
  - サーキットマップ上に速度をヒートマップで表示
  - ドライバー間の速度差を視覚的に把握

#### 3. レースポジション推移
- **ポジションチャート**
  - ラップごとの順位変動を線グラフで表示
  - オーバーテイク発生箇所の強調表示
- **ギャップタイム表示**
  - トップとのギャップ、前車とのギャップの推移

### 予選（Qualifying）分析
#### 1. セクションごとの速度可視化
- **Q1/Q2/Q3別の最速ラップ分析**
  - 各セッションでの最速ラップのセクタータイム比較
  - パープルセクター（最速セクター）の可視化
- **ドライバー比較**
  - 複数ドライバーのセクタータイムを並べて比較
  - どのセクターでタイムを失ったかを明確化
- **テレメトリーデータ**（APIで利用可能な場合）
  - 速度、スロットル、ブレーキポイントの比較
  - ギアチェンジポイントの可視化

### 共通機能
- **レース/セッション選択**
  - シーズン、グランプリ、セッション（FP1/FP2/FP3/予選/本戦/スプリント）の選択
- **ドライバー/チームフィルタリング**
  - 特定のドライバーやチームに絞った分析
  - お気に入りドライバー登録機能
- **データエクスポート**
  - グラフのPNG/SVGエクスポート
  - 分析データのCSVダウンロード
- **レスポンシブデザイン**
  - PC、タブレット、スマートフォンでの閲覧対応

## 技術スタック
- **フロントエンド**: React 18 + TypeScript
- **ルーティング**: React Router v7
- **ビルドツール**: Vite
- **可視化ライブラリ**: Recharts / D3.js
- **状態管理**: Zustand
- **データフェッチング**: TanStack Query (React Query)
- **スタイリング**: Tailwind CSS
- **HTTP クライアント**: Axios
- **デプロイ**: Vercel

## 利用するAPI
- **Open F1**
  - URL: https://openf1.org/
  - 提供データ:
    - リアルタイムおよび過去のレースデータ
    - ラップタイム、セクタータイム
    - ドライバーポジション
    - ピットストップ情報
    - カーテレメトリー（速度、DRS、タイヤ情報など）
    - 天候情報
    - ラジオ通信（一部）

## 今後の拡張案
- **AI予測機能**: 過去データから勝率やポジション予測
- **ソーシャル機能**: ユーザー間でのデータ共有や議論
- **アラート機能**: 特定の条件（オーバーテイク、ファステストラップなど）発生時の通知
- **3Dサーキット表示**: 3Dモデルでのデータ可視化
- **比較分析**: 異なるレース間でのパフォーマンス比較
